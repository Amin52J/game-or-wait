## GameOrWait v1.0.0 — Windows Desktop App

AI-powered game recommendation engine based on your personal taste, now available as a native Windows desktop application.

### Downloads

| File | Format | Description |
|------|--------|-------------|
| `GameOrWait_1.0.0_x64-setup.exe` | NSIS Installer | **Recommended** — Standard setup wizard with install/uninstall support |
| `GameOrWait_1.0.0_x64_en-US.msi` | MSI Installer | Enterprise-friendly Windows Installer package |

### Installation Instructions

#### Option 1: NSIS Installer (Recommended)
1. Download `GameOrWait_1.0.0_x64-setup.exe` from the assets below
2. Run the installer — Windows SmartScreen may show a warning since the app is not code-signed; click "More info" then "Run anyway"
3. Follow the setup wizard to choose your install location
4. Launch GameOrWait from the Start Menu or desktop shortcut

#### Option 2: MSI Installer
1. Download `GameOrWait_1.0.0_x64_en-US.msi` from the assets below
2. Double-click the `.msi` file to start installation
3. Follow the prompts to complete the setup
4. Launch GameOrWait from the Start Menu

### System Requirements
- **OS**: Windows 10 (64-bit) or later
- **Runtime**: WebView2 (pre-installed on Windows 10 21H2+ and Windows 11)

### Uninstalling
- Open Settings > Apps > search for "GameOrWait" > Uninstall
- Or use Add/Remove Programs from the Control Panel
